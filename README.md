# BankManagementSystem_JDBC
It is a Java application with JDBC features to persist the data 

It is a consoled based application with DB to Persist the data with help of JDBC features 

To run this application we need: 
- Java 8+ version
- mysql-connector-j-9.4.0.jar
- needs to provide the environment variables(USER, PASSWORD, URI) for Mysql credientals and url to connect DB.

Features:

1) Show All Accounts
2) Create New Account
3) Account Details
4) Check Balance
5) Withdraw Amount
6) Credit Amount
7) Transfer Money
8) Update Name
9) Update Phone number
